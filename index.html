Weather Visualization Dashboard (Latitude)

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplis the analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included" temperature, cloudiness, wind speed, and humidity. this site provides the source data and visualizations created as part fo the analysis, as well as explainations and descriptions fo any trends and correlations witnessed.
